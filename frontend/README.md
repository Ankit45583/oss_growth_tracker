# 🚀 GitHub Developer Growth Platform (OSS Tracker)

A full-stack MERN application that transforms GitHub activity into meaningful insights and a gamified developer experience.

---

## 🔥 Features

- 🔐 GitHub OAuth Login
- 📊 Dashboard with commits, repos, and activity stats
- 📂 Repository Explorer with filters
- 🏆 Leaderboard (based on developer activity)
- 🔥 Streak Tracking System _(in progress)_
- 🏅 Badge & Achievement System _(in progress)_
- 📈 Growth Insights _(in progress)_
- 🎴 GitHub Wrapped Shareable Card _(planned)_

---

## 🧠 Project Idea

GitHub already shows raw data like commits and repositories, but this platform focuses on:

- Converting data into **insights**
- Tracking **developer growth**
- Adding **gamification** (streaks, badges, leaderboard)
- Making progress **shareable**

---

## 🛠️ Tech Stack

**Frontend:**

- React.js
- Tailwind CSS

**Backend:**

- Node.js
- Express.js

**Database:**

- MongoDB

**Other:**

- GitHub OAuth API
- Cloudinary (for image hosting)
- Puppeteer (for dynamic card generation)
- Node-cron (for scheduled jobs)

---

## ⚙️ Installation

### 1. Clone the repo

```bash
git clone https://github.com/your-username/oss_growth_tracker.git
cd oss_growth_tracker
```

### 2. Backend setup

```bash
cd backend
npm install
npm run dev
```

### 3. Frontend setup

```bash
cd frontend
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file in backend:

```env
MONGO_URI=your_mongodb_uri
GITHUB_CLIENT_ID=your_client_id
GITHUB_CLIENT_SECRET=your_client_secret

CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
```
